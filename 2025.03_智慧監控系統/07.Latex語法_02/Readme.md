## 3、數學符號	LaTeX 語法<br>

| 數學符號 | LaTeX 語法 |
| -------  | ---------|
| 加法	   | +        |
| 減法	   | -        |
| 乘法	   | \times 或 \cdot |
| 除法	   | \div 或 \frac{a}{b} |
| 根號	   | \sqrt{x} 或 \sqrt[n]{x} |
| 等號	   | =                       |
不等號	\neq, \leq, \geq 等
